# ✨ Features

## 🎮 Core Gameplay

* Easy, Medium, and Hard difficulty presets
* Fully customizable game setup
* Adjustable number range system
* Adjustable attempts system
* Intelligent win/loss handling
* Guess feedback system
* Hint system
* Performance-based scoring

---

## 💰 Coin Reward System

Earn coins based on performance.

The fewer guesses used, the larger the reward.

**Reward Range:**

* Maximum Reward → **500 Coins**
* Minimum Reward → **20 Coins**

Includes a live **Payout Preview Window** showing rewards by guess count.

---

## 🛒 Icon Store

Unlock collectible player icons using earned coins.

Features include:

* 30+ unlockable collectables
* Automatic ownership tracking
* Active icon selection
* Persistent purchases
* Saved player customization

---

## 🏅 Achievement System

Progress through a large achievement ladder as your coin bank grows.

Features:

* Auto-unlocking achievements
* 20+ achievement tiers
* Persistent unlock saving
* Highest achievement display

---

## 👤 Player Profile System

Personalize your game experience.

Features:

* Custom player name
* Persistent save system
* Active icon display
* Saved profile loading on refresh

---

## 📜 Guess History Tracking

Every round includes a detailed guess history display.

Shows:

* Previous guesses
* Higher / Lower indicators
* Correct guess tracking
* Automatic reset each new game

---

## 📊 Career Statistics

Track long-term progress across all game modes.

Statistics include:

* Wins
* Losses
* Win percentage
* Mode-specific records
* High score tracking

Supported modes:

* Easy
* Medium
* Hard
* Custom

---

## ⚙️ Game Setup System

Dedicated setup modal allowing players to configure gameplay.

Options include:

* Player Name
* Difficulty Presets
* Custom Maximum Range
* Custom Attempt Count

Rules enforced:

* Minimum range permanently locked to **1**
* Maximum range minimum enforced at **50**
* Custom attempts capped at **15**

---

## 🔒 Protected Reset System

The game includes a multi-stage protected reset workflow.

Safety features include:

* Warning confirmation screen
* Final confirmation screen
* Required typed verification
* Full save overwrite protection

Reset restores default game state including:

* Coin Bank
* Purchases
* Achievements
* Player Icon
* Player Name
